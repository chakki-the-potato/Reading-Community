# 책책책 - 독서 모임 관리

매일 10분 독서 인증과 월 1권 완독을 목표로 하는 독서 모임 관리 웹앱입니다. 카카오톡 채팅 내보내기를 파싱하여 인증 기록을 자동으로 생성하고, 벌금을 계산합니다.

## 주요 기능

- **매일 독서 인증** - 사진 증거 + 3줄 이상 감상문으로 인증
- **인증 기록 조회** - 분기별 인증 현황을 테이블로 확인
- **벌금 자동 계산** - 미인증 일수에 따른 벌금 자동 산출
- **납부 현황 관리** - 멤버별 벌금 납부 여부 추적
- **카카오톡 파싱** - PC/Android/iOS 내보내기 형식 모두 지원

## 기술 스택

| 구분 | 기술 |

## 시작하기

### 웹페이지

`index.html`을 웹 서버를 통해 열면 인증 기록을 조회할 수 있습니다.

### 인증 확인 스크립트

카카오톡 채팅 내보내기 파일을 파싱하여 인증 기록을 생성합니다.

```bash
python3 check_verification.py \
  --members members.csv \
  --start 2026-02-01 \
  --end 2026-02-15 \
  --output records/2026/02/verification_record.csv \
  --label "2월 1분기" \
  chat_export.txt
```

#### 옵션

| 옵션 | 설명 |
|------|------|
| `chat_files` | 카카오톡 채팅 내보내기 파일 (.txt 또는 .csv) |
| `--members, -m` | 멤버 목록 CSV 파일 (필수) |
| `--start, -s` | 시작일 YYYY-MM-DD (필수) |
| `--end, -e` | 종료일 YYYY-MM-DD (필수) |
| `--output, -o` | 출력 CSV 경로 (선택) |
| `--exclude, -x` | 인증 제외일 (선택) |
| `--label, -l` | 기록 라벨, 예: "2월 1분기" (선택) |

### 인증 조건

하루 인증이 유효하려면 같은 날(오전 5시 기준) 아래 두 가지를 모두 충족해야 합니다:

1. **사진** - "사진", "Photo", "동영상" 등의 메시지
2. **감상문** - 첫 줄에 '읽' 포함, 최소 3줄 이상

## 벌금 규칙

| 미인증 일수 | 벌금 |
|-------------|------|
| 1~5일 | 일당 2,000원 (최대 10,000원) |
| 6일 이후 | 일당 1,000원 추가 |

예시: 10일 미인증 = 10,000원 + (5 x 1,000원) = **15,000원**
